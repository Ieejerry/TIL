# 데이터 스트럭쳐란?

현실을 프로그래밍적으로 **표현**하는 것

</br>

### Tree (나무)

예시) 조직도, 디렉토리

</br>

### Set (집합)

예시) 스터디 그룹별 참여자

</br>

### Graph

사용 예) 지도(최적의 경로)

</br>

## 데이터 스트럭쳐의 목적

**큰** 데이터를 효율적으로 관리하는 것

예시) 100권이 넘는 책을 분류(주제 및 알파벳별, 등)하여 책을 빠르고 쉽게 찾아낼 수 있다.

</br>

## 데이터 스트럭쳐는 정리정돈의 진화

**데이터 스트럭쳐는 데이터를 효율적으로 저장하고 꺼내기 쉽게 만드는 것**

**꺼내쓰기 쉽게 데이터를 잘 정리 정돈하는 것**
